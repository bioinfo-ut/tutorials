## Programmid mambas
Paljud programmid on saadavad läbi klastri, kuid need vajavad esmalt laadimist.

Olemasolevad moodulid:
```
module avail
```
Mooduli python-3.6.3 laadimine:
```
module load python-3.6.3
```
